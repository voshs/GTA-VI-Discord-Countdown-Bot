GTA VI Discord Countdown Bot



A simple Discord bot that counts down to the release of GTA VI.



!\[GTA VI Discord Bot](https://raw.githubusercontent.com/voshs/GTA-VI-Discord-Countdown-Bot/main/Screenshot.jpg)



Features

\- !gta6 command to show time remaining

\- Updates the bot status with a live countdown



Requirements

\- Python 3.10 or newer

\- discord.py



Setup

1\. Install dependencies:

&nbsp;  pip install discord.py



2\. Set your Discord bot token as an environment variable:

&nbsp;  DISCORD\_TOKEN=your\_token\_here



3\. Run the bot:

&nbsp;  python bot.py
